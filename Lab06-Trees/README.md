# Lab 06 - Trees

**Course:** Data Structures and Algorithms – IT1170  
**Practical:** 6 – Trees (Binary Search Tree)

## Features Implemented

- Node class with `empNo` and `name`
- `displayNode()` method
- Binary Search Tree operations:
  - `insert()`
  - `find()` (non-recursive)
  - `findRecursive()` (recursive - as specifically asked)
  - In-Order, Pre-Order, Post-Order traversals
  - `deleteAll()` method

- All 10 employees inserted as per lab sheet
- User can search any employee number
- Demonstrates all required traversals and deletion

**Note:** This is a Binary Search Tree (BST) implementation since `insert` places smaller values to the left and larger to the right.

Run `EmployeeTree.java` in IntelliJ to test all functionalities.
