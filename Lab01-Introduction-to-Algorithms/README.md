
## Objectives
- Understand the basics of algorithms and their importance.
- Learn how to write pseudocode.
- Implement simple algorithms using Java in IntelliJ.

## Files Included

| File                              | Description |
|-----------------------------------|-----------|
| `Exercise1-Algorithms.txt`        | Real-life algorithm examples + algorithms for max of 3 numbers and average of N numbers |
| `Pseudocode-Maximum.txt`          | Pseudocode to find maximum of three numbers |
| `Pseudocode-Average.txt`          | Pseudocode to find average of N numbers |
| `FindMaximum.java`                | Java program to find maximum of three numbers (user input) |

## Exercise Summary

**Exercise 1:** Identifying Algorithms
- Two real-life examples similar to navigation system
- Algorithm for navigation system
- Algorithm to find maximum of three numbers
- Algorithm to find average of N numbers

**Exercise 2:** Writing Pseudocodes
- Pseudocode for maximum of three numbers
- Pseudocode for average of N numbers

**Exercise 3:** Converting Pseudocode to Java
- Java implementation of finding maximum of three numbers with user input

## How to Run

1. Open `FindMaximum.java` in IntelliJ IDEA.
2. Run the program.
3. Enter three numbers when prompted.
4. The program will display the maximum number.
